# PlataformaEducativa

Plataforma Educativa con Angular y API de OpenAI

## Descripción

Este proyecto es una plataforma educativa basada en Angular que utiliza la API de OpenAI para proporcionar experiencias de aprendizaje interactivas y personalizadas. La plataforma se enfoca en brindar a los usuarios la capacidad de aprender de manera efectiva y eficiente, utilizando tecnologías de vanguardia y técnicas de inteligencia artificial.

La integración de la API de OpenAI permite a los usuarios interactuar con la plataforma utilizando lenguaje natural y recibir respuestas inteligentes y contextualmente relevantes. La plataforma es capaz de responder preguntas, proporcionar explicaciones detalladas, ofrecer ejercicios prácticos y ofrecer retroalimentación en tiempo real.

## Características

- **Interfaz de Usuario Amigable:** La plataforma cuenta con una interfaz de usuario intuitiva y fácil de usar, diseñada para ofrecer una experiencia de aprendizaje agradable y sin complicaciones.

- **Módulo de Aprendizaje Personalizado:** La plataforma utiliza la API de OpenAI para adaptar el contenido educativo según las necesidades y preferencias de cada usuario. Esto garantiza que el aprendizaje sea personalizado y se ajuste a los objetivos individuales.

- **Sistema de Preguntas y Respuestas:** Los usuarios pueden hacer preguntas utilizando lenguaje natural y recibir respuestas detalladas y contextualmente relevantes proporcionadas por la API de OpenAI.

- **Generación de Ejercicios Prácticos:** La plataforma es capaz de generar ejercicios prácticos en función de los conceptos aprendidos, permitiendo a los usuarios poner en práctica sus conocimientos y recibir retroalimentación instantánea.

- **Seguimiento del Progreso:** La plataforma realiza un seguimiento del progreso de cada usuario, proporcionando estadísticas y métricas para evaluar el rendimiento y el crecimiento en el aprendizaje.

## Instalación

1. Clona este repositorio en tu máquina local: https://github.com/nmedinaes/educational-platform-backend
2. Navega hasta el directorio raíz del proyecto: cd educational-platform-backend
3. Instala las dependencias utilizando npm (Node Package Manager): npm install

## Configuración

Antes de ejecutar la aplicación, debes configurar la API de OpenAI. Asegúrate de tener una clave de API válida y sigue estos pasos:

1. Crea un archivo de configuración para las variables de entorno: touch .env
2. Abre el archivo `.env` en tu editor de texto y agrega la siguiente variable de entorno: OPENAI_API_KEY=tu-clave-de-api

Asegúrate de reemplazar `tu-clave-de-api` con tu propia clave de API de OpenAI.

## Uso

1. Inicia la aplicación Angular: ng serve
2. Abre tu navegador web y visita `http://localhost:4200` para acceder a la plataforma educativa.
3. Explora las diferentes funcionalidades de la plataforma, realiza preguntas, completa ejercicios y sigue tu progreso.
